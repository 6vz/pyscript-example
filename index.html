Python, in HTML, is that even possible?


It turns out that it does! You can use Python in HTML - it may look ridiculous, but it's functional. Using the PyScript plugin, you can execute Python code in HTML. Just use the py-script tag

Press Ctrl + Shift + I (windows and mac), or Cmd + Opt + I (mac) to open the developers Tools, and see Scripts and Action


Some basic Examples...

Input:
print('Hello World!')
Output: print('Hello World!')

Input:
name = "Alexander"
print("Hello " + name + "!")
Output: name = "Alexander" print("Hello " + name + "!")

Input:
name = "Vadim"
if name == "Vadim":
print("Hello " + name + "!") else:
print("Hello Stranger!")
Output: name = "Vadim" if name == "Vadim": print("Hello " + name + "!") else: print("Hello Stranger!")

Input:
a = 2
b = 3
print(a+b)
Output: a = 2 b = 3 print(a+b)

Input:
import platform
print(platform.python_version())
Output: import platform print(platform.python_version())
